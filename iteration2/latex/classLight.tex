\subsection{Detailed Description}
Class representing an mobile \hyperlink{classLight}{Light} within the \hyperlink{classArena}{Arena}. 

\hyperlink{classArena}{Arena} sets the time and calls methods of this class upon timesteps and collisions.

Lights move and upon any collisions reverse in an arc determined by flags and timing of the simulation.
